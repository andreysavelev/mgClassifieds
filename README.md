ngClassifieds!
===================

This application based on Angular 1.5

First you need install dependencies:

``` bash
$ npm install
```

Next run the server

``` bash
$ npm run serve
```

Then open this url in your favorite browser [http://localhost:3000/](http://localhost:3000/)
